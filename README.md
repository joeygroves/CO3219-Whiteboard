# CO3219-Whiteboard
This application is built using Node.js 

## Installation
First clone the git repository

```bash
git clone https://github.com/joeygroves/CO3219-Whiteboard
```

Go into the directory

```bash
cd CO3219-Whiteboard
```

Then
```bash
npm install
```

To run the application itself
```bash
node server.js
```
